## Математическая модель

### Основные формулы

**Полярный момент инерции:**

```
J = (π × d⁴) / 32
```

**Базовый модуль упругости:**

```
G = (M × L) / (J × θ)
где θ в радианах
```

**Эффективный модуль:**

```
В упругой области: G_eff = k × G
В пластичной области: G_eff = k × G₀ × (θ_разр - θ) / (θ_разр - θ_пред)
```

### Коэффициенты материалов

| Материал | k    | θ_пред | θ_разр |
| -------- | ---- | ------ | ------ |
| Сталь    | 1.0  | 15°    | 30°    |
| Чугун    | 0.95 | 10°    | 20°    |
| Дерево   | 0.80 | 8°     | 16°    |
