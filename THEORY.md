## Введение

Модуль упругости второго рода (модуль сдвига) **G** является важнейшей механической характеристикой материала, определяющей его способность сопротивляться деформации сдвига при кручении.

### Цели лабораторной работы:

1. Экспериментально определить модуль сдвига для различных материалов
2. Исследовать зависимость модуля упругости от угла поворота
3. Определить пределы упругости и разрушения для каждого материала
4. Сравнить экспериментальные данные с теоретическими значениями

## Теоретические основы

### Закон Гука при кручении

При кручении стержня круглого сечения между крутящим моментом **M** и углом закручивания **θ** существует линейная зависимость (в пределах упругости):

```
M = G · J · θ / L
```

где:
- **M** — крутящий момент (Н·мм)
- **G** — модуль сдвига (МПа)
- **J** — полярный момент инерции сечения (мм⁴)
- **θ** — угол закручивания (радианы)
- **L** — длина образца (мм)

### Полярный момент инерции

Для круглого сплошного сечения диаметром **d**:

```
J = π · d⁴ / 32
```

Физический смысл: характеризует жесткость сечения при кручении.

### Полярный момент сопротивления

```
Wp = π · d³ / 16
```

Используется для расчета касательных напряжений:

```
τ = M / Wp
```

## Математическая модель

### 1. Расчет базового модуля упругости

Из закона Гука при кручении выражаем модуль сдвига:

```
G = (M · L) / (J · θ)
```

Подставляя выражение для полярного момента инерции:

```
G = (M · L · 32) / (π · d⁴ · θ)
```

где **θ** должен быть выражен в радианах:

```
θ (рад) = θ (град) · π / 180
```

### 2. Эффективный модуль упругости

Учитывая особенности различных материалов, вводится понятие эффективного модуля:

```
G_эфф = k · G
```

где **k** — коэффициент материала:

| Материал | k    | Физический смысл |
|----------|------|------------------|
| Сталь    | 1.00 | Эталонный материал |
| Чугун    | 0.95 | Более хрупкий, меньшая вязкость |
| Дерево   | 0.80 | Анизотропия, неоднородность структуры |

### 3. Области деформации

#### Упругая область (θ ≤ θ_пред)

В этой области материал следует закону Гука:

```
G_эфф = k · G = k · (M · L) / (J · θ)
```

Деформация полностью обратима.

#### Пластическая область (θ_пред < θ < θ_разр)

При превышении предела упругости начинается пластическая деформация. Модуль упругости линейно снижается:

```
G_эфф = k · G₀ · (θ_разр - θ) / (θ_разр - θ_пред)
```

где:
- **G₀** — модуль при пределе упругости
- **θ_пред** — угол предела упругости
- **θ_разр** — угол разрушения

#### Область разрушения (θ ≥ θ_разр)

```
G_эфф = 0
```

Материал теряет несущую способность.

### 4. Характерные углы для материалов

| Материал | θ_пред (°) | θ_разр (°) | Обоснование |
|----------|-----------|-----------|-------------|
| Сталь    | 15        | 30        | Высокая пластичность |
| Чугун    | 10        | 20        | Хрупкий материал |
| Дерево   | 8         | 16        | Анизотропия волокон |

## Физическое обоснование

### Механизм деформации при кручении

1. **Сдвиговые деформации**: Поперечные сечения стержня поворачиваются относительно друг друга
2. **Касательные напряжения**: Возникают напряжения τ, перпендикулярные радиусу сечения
3. **Деформация волокон**: Максимальны на внешней поверхности, равны нулю на оси

### Зависимость касательных напряжений от радиуса

```
τ(r) = (M · r) / J
```

где:
- **r** — расстояние от оси стержня
- максимальное напряжение достигается на поверхности: r = d/2

### Связь между γ и θ

Относительный сдвиг γ связан с углом закручивания:

```
γ = (r · θ) / L
```

А связь между τ и γ определяется модулем сдвига:

```
τ = G · γ
```

## Методика расчета

### Алгоритм определения модуля упругости

```
ВХОД:
  - Материал образца (Сталь, Чугун, Дерево)
  - Длина образца L (мм)
  - Диаметр образца d (мм)
  - Крутящий момент M (Н·мм)
  - Угол поворота θ (градусы)

ШАГ 1: Получить характеристики материала
  k = коэффициент материала
  θ_пред = предел упругости (град)
  θ_разр = угол разрушения (град)

ШАГ 2: Определить область деформации
  ЕСЛИ θ ≤ θ_пред:
    Область = "Упругая"
  ИНАЧЕ ЕСЛИ θ < θ_разр:
    Область = "Пластическая"
  ИНАЧЕ:
    Область = "Разрушение"

ШАГ 3: Рассчитать полярный момент инерции
  J = π · d⁴ / 32

ШАГ 4: Рассчитать модуль упругости
  ЕСЛИ Область == "Упругая":
    θ_рад = θ · π / 180
    G = (M · L) / (J · θ_рад)
    G_эфф = k · G
  
  ИНАЧЕ ЕСЛИ Область == "Пластическая":
    θ_пред_рад = θ_пред · π / 180
    G₀ = (M · L) / (J · θ_пред_рад)
    G_эфф = k · G₀ · (θ_разр - θ) / (θ_разр - θ_пред)
  
  ИНАЧЕ:
    G_эфф = 0

ШАГ 5: Проверка корректности
  Сравнить G_эфф с типичными значениями для материала

ВЫХОД:
  - G_эфф (МПа) — эффективный модуль упругости
  - Область деформации
  - Предупреждения (если есть)
```

### Проверка результатов

Типичные значения модуля сдвига:

| Материал | Диапазон G (МПа) | Типичное значение |
|----------|------------------|-------------------|
| Сталь    | 75,000 - 85,000  | 80,000 МПа        |
| Чугун    | 30,000 - 50,000  | 40,000 МПа        |
| Дерево   | 500 - 2,000      | 1,000 МПа         |

## Примеры расчетов

### Пример 1: Сталь в упругой области

**Дано:**
- Материал: Сталь
- L = 100 мм
- d = 10 мм
- M = 1000 Н·мм
- θ = 10° (< 15° — упругая область)

**Решение:**

1. Коэффициент материала: k = 1.0

2. Полярный момент инерции:
   ```
   J = π · 10⁴ / 32 = 981.75 мм⁴
   ```

3. Угол в радианах:
   ```
   θ = 10° · π / 180 = 0.1745 рад
   ```

4. Базовый модуль:
   ```
   G = (1000 · 100) / (981.75 · 0.1745) = 58,177 МПа
   ```

5. Эффективный модуль:
   ```
   G_эфф = 1.0 · 58,177 = 58,177 МПа
   ```

**Ответ:** G_эфф = 58,177 МПа ✓ (в пределах нормы для упругой деформации)

---

### Пример 2: Чугун в пластической области

**Дано:**
- Материал: Чугун
- L = 150 мм
- d = 12 мм
- M = 1500 Н·мм
- θ = 15° (> 10° — пластическая область)

**Решение:**

1. Коэффициент материала: k = 0.95
   Предел упругости: θ_пред = 10°
   Угол разрушения: θ_разр = 20°

2. Полярный момент инерции:
   ```
   J = π · 12⁴ / 32 = 2,035.75 мм⁴
   ```

3. Модуль при пределе упругости:
   ```
   θ_пред_рад = 10° · π / 180 = 0.1745 рад
   G₀ = (1500 · 150) / (2035.75 · 0.1745) = 633,028 МПа
   ```

4. Эффективный модуль в пластической области:
   ```
   G_эфф = 0.95 · 633,028 · (20 - 15) / (20 - 10)
   G_эфф = 0.95 · 633,028 · 0.5 = 300,688 МПа
   ```

**Ответ:** G_эфф = 300,688 МПа  
⚠️ Материал находится в пластической области, возможны остаточные деформации

---

### Пример 3: Дерево на пределе разрушения

**Дано:**
- Материал: Дерево
- L = 100 мм
- d = 10 мм
- M = 800 Н·мм
- θ = 14° (близко к 16° — предел разрушения)

**Решение:**

1. Коэффициент материала: k = 0.80
   Предел упругости: θ_пред = 8°
   Угол разрушения: θ_разр = 16°

2. Полярный момент инерции:
   ```
   J = 981.75 мм⁴
   ```

3. Модуль при пределе упругости:
   ```
   θ_пред_рад = 8° · π / 180 = 0.1396 рад
   G₀ = (800 · 100) / (981.75 · 0.1396) = 583,841 МПа
   ```

4. Эффективный модуль:
   ```
   G_эфф = 0.80 · 583,841 · (16 - 14) / (16 - 8)
   G_эфф = 0.80 · 583,841 · 0.25 = 116,768 МПа
   ```

**Ответ:** G_эфф = 116,768 МПа  
⚠️⚠️ Материал близок к разрушению! Риск необратимых повреждений

## Анализ экспериментальных данных

### Определение модуля на линейном участке

Для набора экспериментальных точек (Mᵢ, θᵢ) модуль определяется по методу наименьших квадратов на линейном участке:

```
G = Σ(Mᵢ · L) / (J · Σθᵢ)
```

### Определение предела пропорциональности

Предел пропорциональности τ_пц соответствует моменту отклонения от линейной зависимости M(θ).

### Определение предела текучести

Предел текучести по остаточной деформации 0.3%:

```
γ₀.₃% = 0.003
θ₀.₃% = (γ · L) / r
T₀.₃% = (G · J · θ₀.₃%) / L
τ₀.₃% = T₀.₃% / Wp
```

## Погрешности и точность

### Источники погрешностей:

1. **Геометрические измерения** (± 0.1 мм):
   - Погрешность диаметра влияет как d⁴ (очень критично!)
   - Погрешность длины линейна

2. **Измерение угла** (± 0.5°):
   - Прямо пропорционально влияет на G

3. **Измерение момента** (± 2%):
   - Линейное влияние на G

4. **Неоднородность материала**:
   - Для стали: ± 2-3%
   - Для чугуна: ± 5-8%
   - Для дерева: ± 10-20%

### Оценка полной погрешности:

```
ΔG/G = √[(4·Δd/d)² + (ΔL/L)² + (Δθ/θ)² + (ΔM/M)²]
```

Типичная погрешность: 5-10% для металлов, 15-25% для дерева.

## Практические рекомендации

### Для получения точных результатов:

1. **Тщательно измерить диаметр** в нескольких местах, взять среднее
2. **Провести несколько измерений** при каждом значении момента
3. **Учитывать температуру** (модуль зависит от T)
4. **Проверять центровку** образца в установке
5. **Постепенно увеличивать нагрузку** для наблюдения перехода к пластичности

### Контроль качества результатов:

- ✓ G должен быть положительным
- ✓ G должен находиться в типичном диапазоне для материала
- ✓ В упругой области G ≈ const
- ✓ В пластической области G монотонно убывает

## Выводы

1. **Модуль сдвига G** — фундаментальная характеристика материала, определяющая его поведение при кручении

2. **Упругая область** характеризуется постоянством модуля и обратимостью деформации

3. **Пластическая область** проявляется снижением эффективного модуля и появлением остаточных деформаций

4. **Различные материалы** имеют различные пределы упругости и углы разрушения

5. **Точность расчетов** критически зависит от точности измерения диаметра (влияние ∝ d⁴)
